Convenience composables over Convex's file storage. All auto-imported.

## `useUpload()`

Upload a single file to a Convex storage upload URL:

```vue
<script setup lang="ts">
import { api } from '#backend/api'

const { upload, progress, isUploading } = useUpload({
  generateUploadUrl: api.files.generateUploadUrl,
})

async function onChange(e: Event) {
  const file = (e.target as HTMLInputElement).files?.[0]
  if (file) {
    const storageId = await upload(file)
  }
}
</script>
```

## `useUploadQueue()`

Manage several uploads at once, each with its own status and progress (a deeply reactive queue):

```ts
const { items, enqueue } = useUploadQueue({
  generateUploadUrl: api.files.generateUploadUrl,
})
```

## `useStorageUrl()`

Resolve a stored file's served URL (reactive to the id):

```ts
const url = useStorageUrl({
  getStorageUrl: api.files.getUrl,
  storageId,
})
```

::callout{icon="i-lucide-shield"}
In production, the bundled CSP allows Convex-served storage on `img-src` / `media-src` automatically.
::
