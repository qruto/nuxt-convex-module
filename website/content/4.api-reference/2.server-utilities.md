Auto-imported in Nitro server code — server routes, API handlers, server middleware, and Nitro plugins (a port of `convex/nextjs`). App code that runs during SSR (a page's `useAsyncData`, `<script setup>`) does **not** get these auto-imports: there, import them explicitly from `nuxt-convex-module/server`.

| Name | Description |
|---|---|
| `fetchQuery` | One-shot query over HTTP. |
| `fetchMutation` | One-shot mutation. |
| `fetchAction` | One-shot action. |
| `preloadQuery` | Preload a query for SSR hydration. |
| `preloadedQueryResult` | Extract the value from a preloaded query. |

All four data functions accept a trailing options object after the function args: `{ token?, url?, skipConvexDeploymentUrlCheck? }`. `token` is a JWT for authenticated calls without Better Auth, `url` overrides the deployment URL (defaults to `NUXT_PUBLIC_CONVEX_URL`), and `skipConvexDeploymentUrlCheck: true` allows self-hosted URLs that don't look like `*.convex.cloud`. Full signatures: [`nuxt` reference](/api-reference/reference/nuxt).

```ts [server/api/messages.get.ts]
import { api } from '#backend/api'

export default defineEventHandler(async (event) => {
  const token = getHeader(event, 'authorization')?.replace('Bearer ', '')
  return fetchQuery(api.messages.list, {}, { token })
})
```

### With `@convex-dev/better-auth`

| Name | Description |
|---|---|
| `backendAuth(event, opts?)` | A request-scoped, authenticated helper service (`getToken`, `isAuthenticated`, `fetchAuth*`, `preloadAuthQuery`, and `handler` — the Better Auth route proxy, the Nuxt analog of upstream's `nextJsHandler`). Alias: `convexBetterAuthNuxt`. |

The optional second argument extends `GetTokenOptions` from `@convex-dev/better-auth/utils` with a `convexSiteUrl` override (defaults to `NUXT_PUBLIC_CONVEX_SITE_URL`). Full signatures: [`better-auth/nuxt/server` reference](/api-reference/reference/better-auth/nuxt/server).

Import the `fetch*` / `preload*` helpers directly (outside auto-import contexts) from `nuxt-convex-module/server`; `backendAuth` comes from `nuxt-convex-module/better-auth/server`.
