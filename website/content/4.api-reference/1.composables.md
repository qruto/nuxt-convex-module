Everything below is auto-imported. `useQuery`, `useQueries`, `useMutation`, `useAction`, `usePaginatedQuery`, `useUpload`, `useUploadQueue`, and `useStorageUrl` also have `useConvex*`-prefixed aliases.

## Data

| Name | Description |
|---|---|
| `useQuery` / `useQuery_experimental` | Reactive live query (throwing / result-union). |
| `useQueries` | A dynamic number of live queries from one composable call. |
| `useMutation` | Call a Convex mutation (supports optimistic updates). |
| `useAction` | Call a Convex action. |
| `usePaginatedQuery` / `usePaginatedQuery_experimental` | Cursor pagination. |
| `usePreloadedQuery` | Hydrate a query preloaded on the server with `preloadQuery`. |
| `useConvexConnectionState` | Live WebSocket connection status. |
| `useConvex` | The underlying `ConvexVueClient`. |

## Files

| Name | Description |
|---|---|
| `useUpload` / `uploadFile` | Upload to Convex storage. |
| `useUploadQueue` | Multi-file upload queue. |
| `useStorageUrl` | Resolve a stored file's URL. |
