Auto-imported in any Nitro server route, API handler, or SSR context (a port of `convex/nextjs`).

| Name | Description |
|---|---|
| `fetchQuery` | One-shot query over HTTP. |
| `fetchMutation` | One-shot mutation. |
| `fetchAction` | One-shot action. |
| `preloadQuery` | Preload a query for SSR hydration. |
| `preloadedQueryResult` | Extract the value from a preloaded query. |

### With `@convex-dev/better-auth`

| Name | Description |
|---|---|
| `backendAuth(event)` | A request-scoped, authenticated Convex client. |

Import them directly (outside auto-import contexts) from `@qruto/nuxt-convex/server`.
