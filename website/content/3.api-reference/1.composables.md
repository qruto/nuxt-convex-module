Everything below is auto-imported. Each composable also has a `useConvex*`-prefixed alias.

## Data

| Name | Description |
|---|---|
| `useQuery` / `useQuery_experimental` | Reactive live query (throwing / result-union). |
| `useQueries` | Several live queries over one subscription. |
| `useMutation` | Call a Convex mutation (supports optimistic updates). |
| `useAction` | Call a Convex action. |
| `usePaginatedQuery` / `usePaginatedQuery_experimental` | Cursor pagination. |
| `useConvexConnectionState` | Live WebSocket connection status. |
| `useConvex` | The underlying `ConvexVueClient`. |
